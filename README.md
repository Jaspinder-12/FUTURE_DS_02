# Customer Churn Analysis Dashboard

A Data Science & Analytics internship project focused on analyzing customer churn using Python and Power BI. The project explores customer behavior, identifies churn patterns, and presents business insights through an interactive dashboard.

---

## Project Overview

Customer churn is one of the biggest challenges for subscription-based businesses. This project analyzes customer data to identify factors influencing churn and provides interactive visualizations to support business decision-making.

The dataset was cleaned using Python and visualized in Power BI to create an easy-to-understand dashboard.

---

## Objectives

- Clean and preprocess customer data
- Perform Exploratory Data Analysis (EDA)
- Identify factors contributing to customer churn
- Build an interactive Power BI dashboard
- Generate business insights for customer retention

---

## Tools & Technologies

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Power BI
- CSV Dataset

---

## Dataset

**Dataset:** IBM Telco Customer Churn Dataset

The dataset contains customer information including:

- Customer ID
- Gender
- Senior Citizen
- Contract Type
- Internet Service
- Payment Method
- Monthly Charges
- Total Charges
- Customer Tenure
- Churn Status

---

## Data Cleaning

The following preprocessing steps were performed:

- Converted TotalCharges from text to numeric
- Removed missing values
- Checked duplicate records
- Verified data types
- Exported cleaned dataset for Power BI

---

## Exploratory Data Analysis

EDA was performed using Python to understand customer behavior.

The analysis includes:

- Customer Churn Distribution
- Contract Type Analysis
- Internet Service Analysis
- Payment Method Analysis
- Monthly Charges Distribution
- Customer Tenure Distribution

---

## Power BI Dashboard

The dashboard contains the following KPIs:

- Total Customers
- Churned Customers
- Active Customers
- Churn Rate
- Average Monthly Charges

### Dashboard Visualizations

- Contract Type vs Customer Churn
- Internet Service vs Customer Churn
- Payment Method vs Customer Churn
- Gender vs Customer Churn
- Senior Citizen vs Customer Churn
- Monthly Charges Distribution
- Customer Tenure Distribution

### Interactive Filters

- Gender
- Contract Type
- Internet Service

---

## Key Insights

- Month-to-Month contracts have the highest churn.
- Fiber Optic customers are more likely to churn.
- Customers using Electronic Check have higher churn rates.
- Long-term customers show better retention.
- Monthly charges have an impact on customer churn.
